# VORON2.4-LM_ShaftMOD
voron2.4z轴光轴代替线轨套件
使用8mm光轴代替线轨  
使用LM8UU轴承（17mm长）配合打印件代替MGNH滑块，孔位完全兼容  
建议配合原版GE5C鱼眼轴承mod使用  
安装方法见附件
